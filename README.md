# Campus Exploration System

A Python-based campus exploration system that applies classic data structures and algorithms to location management, path exploration, leaderboard ranking, and reward optimisation.

---

## Overview

Campus Exploration System is a simulation project that models a university campus as a graph and provides multiple management modules for analysing locations, rewards, player rankings, and exploration strategies.

The project demonstrates the practical application of object-oriented programming (OOP) together with fundamental data structures and algorithms.

---

## Features

### Campus Management
- Load campus maps from configuration files
- Build graph-based campus connections
- Manage locations and rewards

### Exploration Module
- Greedy path exploration
- Longest path reward calculation
- Total path difficulty analysis
- Recursive graph traversal

### Leaderboard System
- Player ranking management
- Merge multiple leaderboards
- Merge Sort based ranking

### Location Manager
- Search locations within a desirability range
- Retrieve Top-K desirable locations
- Update location rewards dynamically

### Reward Seeker
- Priority-based reward selection
- Dynamic reward updates
- Max Heap optimisation
